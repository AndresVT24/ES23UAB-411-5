# QueMenges

Proyecto de **Enginyeria del Software (UAB, curs 2025-26)**.

QueMenges es una aplicación que permite a los usuarios acceder a platos y menús preparados por personas individuales, ofreciendo una alternativa a plataformas centradas en grandes cadenas o restaurantes. En la aplicación, cualquier persona puede ofrecer platos y recetas, y los usuarios pueden consultar, buscar, pedir y valorar comidas en tiempo real.

## Descripción del proyecto

QueMenges es una plataforma orientada al área metropolitana de Barcelona que conecta a clientes con cocineros particulares y repartidores.

La aplicación contempla, entre otras, las siguientes funcionalidades:

- Registro e inicio de sesión de usuarios.
- Gestión de perfiles con distintos roles:
  - cliente
  - cocinero
  - repartidor
- Consulta de platos y menús disponibles.
- Búsqueda de platos o cocineros.
- Filtrado según restricciones alimentarias.
- Realización y reserva de pedidos.
- Gestión del envío de pedidos.
- Pago dentro de la aplicación.
- Valoración de pedidos recibidos.
- Funcionalidades adicionales como propuestas de platos, chat y sistema de puntos.

## Perfiles de usuario identificados

- **Cliente**
  - Consulta platos y menús
  - Realiza pedidos
  - Paga pedidos
  - Valora pedidos
- **Cocinero**
  - Publica platos y menús
  - Acepta pedidos
  - Gestiona disponibilidad
- **Repartidor**
  - Gestiona entregas y transporte de pedidos
